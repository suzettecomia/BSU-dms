Help

Is there a file size limit when uploading documents?

- None, but the limit is undertaken when storage is already full. Once full, uploading files are not allowed until the storage is upgraded.

Are all documents in the system accessible on a mobile device?

- Yes, DMS is accessible through mobile devices and files are still downloadable.

What types of documents can be stored?

- DMS has a limited type of document, either Word, Spreadsheets, PDF, or PowerPoint.

Can I sort or group my documents?

- Yes, DMS provides a list of categories so that you can easily sort or group your documents based on its classification.

Can I create my own account?

- No, the administrator will be the one who creates the account for security purposes.

Can I change my password once created?

- Yes, there will be a user profile and passwords are changeable.

What will happen if my request document is still not approved?

- Yes, there will be a user profile and passwords are changeable.
